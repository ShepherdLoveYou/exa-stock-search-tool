# Exa Stock Search - Claude Integration

You have access to a powerful stock market search tool via the Exa Search Skill.

## Available Commands

### When User Asks About Stock Information

If the user asks about:
- Stock news or price updates
- Market trends
- Company information
- Financial analysis
- Market research

**Execute:** Use the search command to get real-time data

### Direct Function Call

You can call the skill with natural language queries:

```
search_stock("What's the latest with Apple stock?")
search_stock("Analyze AI tech companies")
search_stock("Research Microsoft earnings")
```

### Query Types (Auto-Detected)

The system automatically detects and routes queries:

1. **Stock News** - When asking about specific tickers
   - "Latest with AAPL?"
   - "TSLA stock news?"
   - "What's happening with Microsoft?"

2. **Market Analysis** - When asking about sectors/trends
   - "AI stocks trends"
   - "Tech sector analysis"
   - "Growth stock market"

3. **Company Research** - When asking about companies
   - "Research Apple company"
   - "Microsoft information"
   - "Tell me about Tesla"

4. **Deep Research** - When asking for thorough analysis
   - "Deep research on semiconductors"
   - "Comprehensive analysis of AI sector"
   - "Thoroughly research quantum computing"

## How to Use

### For Stock News
```
User: "What's happening with Apple stock?"
You: "I'll search for the latest Apple stock news for you."
[Use: search_stock("What's the latest with Apple stock?")]
```

### For Market Analysis
```
User: "Analyze the AI sector"
You: "Let me research current AI stock trends."
[Use: search_stock("Analyze AI tech stocks")]
```

### For Company Research
```
User: "Tell me about Microsoft"
You: "I'll get current information about Microsoft."
[Use: search_stock("Research Microsoft company")]
```

## Setup Required

### One-time Setup in Your System

If not already set up, direct the user to:

1. Get API key from: https://dashboard.exa.ai/api-keys
2. Add to their local `.env` file:
   ```
   EXA_API_KEY=your_key_here
   ```

### Local Usage

Users can run independently:
```bash
# Direct command
python src/claude_skill.py "What's the latest with Apple?"

# Or via Python
python -m src.examples.claude_skill_demo
```

## Key Features

✅ **Natural Language Understanding** - Understands various ways to phrase queries  
✅ **Intelligent Routing** - Automatically chooses best search type  
✅ **Real-time Data** - Uses Exa's latest web index  
✅ **Fast Responses** - ~1 second search time  
✅ **Efficient Tokens** - Uses highlights instead of full text  
✅ **Flexible Output** - Text or JSON format  

## Search Configuration

Used by default:
- **Type**: `auto` (balanced relevance and speed)
- **Results**: 10 per search
- **Content**: Highlights (efficient)
- **Speed**: ~1 second

Customizable per query.

## Error Handling

If search fails:
- Check that `.env` has `EXA_API_KEY` set
- Verify internet connection
- Try a simpler query
- Check API quota at dashboard.exa.ai

## Limitations

- Requires API key configuration
- API rate limits apply
- Results depend on Exa's web index
- Financial data may have slight delays
- Real-time prices may not be available
